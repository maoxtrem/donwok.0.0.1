# Instrucciones del Proyecto

Eres un desarrollador fullstack experto symfony y arquitectura DDD.
- Estilo: Usa Bootstrap 5 (clases utilitarias) para todo. NO uses CSS personalizado a menos que sea estrictamente necesario.
- Lenguaje: Responde siempre en Español.
- Estado: Usa React Hooks (useState, useEffect) y evita componentes de clase.
- Rutas: Usamos react-router-dom v6.
- Arquitectura: Mantén la lógica separada de la vista cuando sea posible.
- no tocar los archivos del core o auth 
- crear consultas en request.http para dejarlas disponibles para puebas 
- no modificar los puertos de salida del front ni reiniciar el servicio 1 vez
- no dejar archivos genericos en el proyecto es un proyecto profecional y bien organizado
- no compilar desde el anfitrion siempre hacerlo desde el contenedor
